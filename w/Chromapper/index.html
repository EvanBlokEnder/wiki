Chromapper Tutorial

Everything you need to know about mapping in Chromapper

Getting Started with Chromapper

1

Installation and Setup

Download Chromapper from the official website and install it on your computer. Make sure your system meets the minimum requirements.

2

Creating Your First Project

Launch Chromapper and create a new project. Select the appropriate settings for your map including BPM, time signature, and difficulty level.

New Project Dialog

Set your BPM and other initial settings when creating a new project

3

Importing Your Audio File

Select the sound file, then press save and select your music file. Chromapper will analyze the file and prepare it for mapping.

PRO TIP

Use high quality audio files (preferably OGG or EGG) for the best results. MP3s with low bitrate can cause timing issues.

4

Basic Interface Overview

Familiarize yourself with the main components of Chromapper's interface:

  • Timeline: Where you place your notes
  • Toolbar: Contains all mapping tools
  • Preview Window: Shows how your map will look in-game
  • Properties Panel: Adjust note properties and map settings

Chromapper Keybinds

Master these keyboard shortcuts to speed up your mapping workflow.

General Shortcuts

Action Keybind Description
Save Project Ctrl + S Saves your current project
Undo Ctrl + Z Undo last action
Redo Ctrl + Y Redo last undone action
Play/Pause Space Toggle playback

Mapping Tools

Tool Keybind Description
Select Tool 1 Select and move notes
Note Tool 2 Place new notes
Delete Tool 3 Remove notes
Slice Tool 4 Split long notes

Timeline Navigation

Action Keybind Description
Zoom In + Zoom in on timeline
Zoom Out - Zoom out on timeline

PRO TIP

You can customize these keybinds in the Settings menu under "Keyboard Shortcuts" to match your personal preferences.

Mapping Basics

1

Placing Your First Notes

Select the Note Tool (2) and click on the timeline where you want to place notes. Use your audio waveform and beat markers as guides.

Place notes aligned with the beat markers and musical elements

2

Understanding Note Types

Chromapper has several note types:

  • Regular Notes: Basic notes players need to hit
  • Hold Notes: Long notes that require holding
  • Chain Notes: Connected notes that flow together
  • Special Notes: Unique effects like explosions
3

Timing and Beat Alignment

Proper timing is crucial for a good map. Use these techniques:

  • Enable the metronome while mapping (M)
  • Use the beat snap feature to align notes perfectly
  • Adjust your global offset if notes feel consistently early/late

Advanced Mapping Techniques

Pattern Theory

Learn to create interesting and playable patterns:

  • Vary your patterns to match the music's intensity
  • Use mirroring and rotation for symmetrical patterns
  • Create flow between patterns for smooth gameplay

Advanced Timing Techniques

Take your timing to the next level:

  • Use BPM changes for songs with tempo variations
  • Create custom time signatures for complex rhythms
  • Apply micro-adjustments for perfect sync

Effect Mapping

Enhance your maps with visual effects:

  • Add light effects synchronized with drops
  • Use environmental changes for big moments
  • Create custom particle effects

Workflow Tips

Efficient Mapping Strategies

Streamline your mapping process:

  • Map in passes - first rhythm, then patterns, then polish
  • Use bookmarks (B) to mark important sections
  • Take breaks to maintain fresh perspective

Testing and Playtesting

Ensure your map plays well:

  • Test frequently as you map
  • Get feedback from other mappers
  • Watch replays to identify problem areas

Publishing Your Map

Share your creation with the world:

  • Create a complete package with all necessary files
  • Write clear map description and credits
  • Submit to the appropriate community platforms
